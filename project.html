voice command board

___< Components >___
1. Arduino Uno
2. 4-Channel Relay Module
3. HC-05 Bluetooth Module
4. Mobile with Voice Assistant App (e.g. Google Assistant + IFTTT + Arduino Bluetooth Controller)
5. Jumper Wires
6. 230V AC Bulb & Fan
7. Power Supply for Arduino
8. Breadboard (optional)
---<<<(Step by Step Process:)>>>---
1. Wiring Diagram Overview: Arduino को Relay से कनेक्ट करो (D2, D3, D4, D5 to IN1, IN2, IN3, IN4) HC-05 Bluetooth: TX to Arduino RX, RX to Arduino TX Relay का NO और COM को AC Switch Line से जोड़ो Bulb या Fan का एक Terminal Relay से, दूसरा Neutral से Arduino को Power दो (5V)

--- 2. Arduino Code:
char data = 0;
void setup()
{ Serial.begin(9600);
pinMode(2, OUTPUT); // Relay 1
pinMode(3, OUTPUT); // Relay 2
pinMode(4, OUTPUT); // Relay 3
pinMode(5, OUTPUT); // Relay 4 }
void loop() {
if (Serial.available()) {
data = Serial.read();
if (data == 'A') digitalWrite(2, HIGH); // Light ON
if (data == 'a') digitalWrite(2, LOW); // Light OFF
if (data == 'B') digitalWrite(3, HIGH); // Fan ON
if (data == 'b') digitalWrite(3, LOW); // Fan OFF }
}

--- 3. Bluetooth Configuration: HC-05 को मोबाइल के साथ पेयर करो कोई Bluetooth Terminal App (जैसे Arduino Bluetooth Controller) इंस्टॉल करो

--- 4. Voice Assistant + IFTTT Setup: IFTTT ऐप डाउनलोड करो नया Applet बनाओ: If This: Google Assistant → Say "Turn on light" Then That: → Android SMS या Webhook → HC-05 को 'A' भेजो

--- 5. Safety Notes: Relay को AC पर संभालकर लगाओ Arduino को USB से Power देना ही बेहतर रहेगा Insulation और Relay Module को Box में रखो
